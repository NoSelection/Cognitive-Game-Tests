Nasıl Oynanır?

1. Oyunu başlatmak için "Yeni Oyun" butonuna tıklayın.

2. Hedef desen yukarıda, sizin deseniniz aşağıdadır.

3. Aşağıdaki desendeki bloklara tıklayarak yerlerini değiştirin.

4. Aşağıdaki desen hedef desenle aynı olduğunda seviyeyi geçersiniz.

5. Zamanınız bittiğinde veya desenleri eşleştiremediğinizde oyun biter.
